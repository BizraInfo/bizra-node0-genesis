API Documentation

Comprehensive RESTful API for user management, authentication, and resource access

Authentication: Most endpoints require authentication. Click the "Authorize" button and enter your Bearer token.

Quick Start: 1. Use POST /auth/register to create an account
2. Use POST /auth/login to get access token
3. Click "Authorize" and paste the access token
4. Try authenticated endpoints like GET /users
