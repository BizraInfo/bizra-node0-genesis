# BIZRA Brand Guide

## Purpose

Concise reference for colors, typography, spacing, and component radii aligned with the dashboard.

## Colors

- Primary: #0C8CE9
- Primary (Dark): #066BB3
- Secondary: #10B981
- Accent: #F59E0B
- Danger: #EF4444
- Success: #22C55E
- Info: #3B82F6
- Background: #0F172A
- Surface: #0B1222
- Card: #101828
- Border: #1F2937
- Text: #E5E7EB
- Text Muted: #9CA3AF

## Typography

- Sans: Inter, Segoe UI, Roboto, Helvetica, Arial, sans-serif
- Mono: JetBrains Mono, Menlo, SFMono-Regular, monospace
- Base Size: 14px (sm: 12px, lg: 16px)
- Weights: 400 / 500 / 700

## Spacing

- xs: 4px, sm: 8px, md: 12px, lg: 16px, xl: 24px, xxl: 32px

## Radii

- sm: 6px, md: 8px, lg: 12px

## Implementation

- Tokens: `src/dashboard/theme/tokens.ts`
- CSS Vars: `src/dashboard/index.css` (prefixed `--brand-*`)
- React Components: Ant Design themed via CSS overrides
