## ?? TROUBLESHOOTING

### **Issue 1: Dashboard doesn't start**

**Symptoms:** `npm run dev` fails or shows errors

**Solutions:**
```powershell
# 1. Clean install
cd C:\BIZRA-NODE0\bizra-dashboard
Remove-Item node_modules -Recurse -Force
Remove-Item package-lock.json -Force
npm install

# 2. Check Node version
node --version  # Should be v18+

# 3. Clear Vite cache
Remove-Item .vite -Recurse -Force
npm run dev
```

### **Issue 2: Synaptic API not responding**

**Symptoms:** API health check fails

**Solutions:**
```powershell
# 1. Check if port 9470 is in use
netstat -ano | findstr :9470

# 2. Kill existing process
taskkill /PID <PID> /F

# 3. Restart API
python node0/synaptic-hyperdrive-api.py

# 4. Verify health
Invoke-WebRequest -Uri http://localhost:9470/health
```

### **Issue 3: WebSocket connection fails**

**Symptoms:** "Offline" status in dashboard

**Solutions:**
```powershell
# 1. Check if WebSocket server is running
# Look for "WebSocket connected" in API logs

# 2. Verify CORS settings
# Check synaptic-hyperdrive-api.py CORS config

# 3. Test WebSocket manually
# Use browser dev tools: Network ? WS tab
```

### **Issue 4: Arabic characters display as ??**

**Symptoms:** ????? shows as ?? in UI

**Solutions:**
```powershell
# 1. Check file encoding (should be UTF-8)
# In VSCode: bottom right ? UTF-8

# 2. Verify font loading
# Check browser dev tools ? Network ? Fonts

# 3. Use English fallback
# Dashboard automatically uses "ahsan" internally
```

### **Issue 5: Build fails with ESBuild error**

**Symptoms:** Build stops with module errors

**Solutions:**
```powershell
# 1. Check for syntax errors
```
